## Quick start

You'll first need to host the API locally (see: [api#quick-start](https://github.com/kanap-shop/api#quick-start))

```sh
# clone repo
$ git clone https://github.com/kanap-shop/api.git

# install dependencies
$ yarn install

# serve on localhost
$ yarn serve

# build
$ yarn build
```
